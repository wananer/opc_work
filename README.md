# 三省六部 - 多Agent协作框架

基于中国古代三省六部制度的AI多Agent协作架构。

## 架构简介

三省六部是一套基于中国古代三省六部制度设计的AI多Agent协作框架，通过分工明确的部门协作机制，让不同部门负责不同类型的任务，实现高效有序的AI协作。
